redis 简介

Redis 是一个开源的使用 C 语言编写的支持持久化的 Key-value 型数据库，并内置提供多种语言的API。被称为数据结构服务器，因为值（value）可以是 字符串(String), 哈希(Map), 列表(list), 集合(sets) 和 有序集合(sorted sets)等类型，应用非常广泛。

redis-cluster 集群搭建

单服务器一主一从

1、如果下载的包是已经 make 过的，或者从其他集群 copy 过来的，则不需要在 make 安装


2、一主一从搭建 需要 redis-3.2.0[已编译的安装包]解压缩到两个不同的目录作为一主、一从，然后分别设置相应 redis 下的 *.conf配置文件

此种搭建方式适合生产大型项目缓存集群的搭建，一个 redis 包作为一个 cluster集群的节点。
redis-server 是支持通过指定配置文件启动的，不同的配置文件对应不同的端口，所以同一个 redis 包 也可以配置多个节点组件集群， 搭建方法可参照博客(链接)

3、在redis的目录下创建rdb目录：mkdir -p /data/crm/crmcache1/redis-3.2.0/rdb，如果是从其他集群拿过来的文件，记得要删除rdb/下的文件

rdb 为 redis 持久化文件存放目录，包括 当前节点在集群的持久化信息rdb`aof，以及记录节点nodes-6379.conf`在cluster集群中的角色，这些信息需要删除重新生成，否则启动结点时会去加载
